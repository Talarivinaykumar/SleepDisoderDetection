# 🚀 Quick Start Guide

## AI Sleep Disorder Prediction System

### ⚡ Running the Application

#### Option 1: Using the Batch File (Easiest)
1. Double-click `run.bat` in the project folder
2. Wait for the server to start
3. Open your browser and go to: `http://localhost:5000`

#### Option 2: Using Command Line
1. Open Command Prompt or PowerShell
2. Navigate to the project folder:
   ```
   cd d:\sleep
   ```
3. Run the application:
   ```
   python app.py
   ```
4. Open your browser and go to: `http://localhost:5000`

### 📱 Using the Application

#### Home Page
- Beautiful landing page with animated doctor illustration
- Click "Start Analysis" to begin prediction
- Click "About Project" to learn more

#### Prediction Page
- Fill in all 8 health metrics:
  - Age (e.g., 35)
  - Gender (Male/Female)
  - Sleep Duration (e.g., 6.5 hours)
  - Stress Level (1-10 scale, e.g., 7)
  - Daily Steps (e.g., 5000)
  - Heart Rate (e.g., 75 bpm)
  - Systolic BP (e.g., 120)
  - Diastolic BP (e.g., 80)
- Click "Analyze Sleep Pattern"
- View your results with personalized recommendations

#### About Page
- Learn about the problem statement
- Understand the ML model
- View technology stack
- See system architecture

### 🛠️ Troubleshooting

**Server won't start?**
- Make sure Python is installed
- Install dependencies: `pip install -r requirements.txt`
- Check if port 5000 is available

**Page not loading?**
- Ensure the Flask server is running
- Check the terminal for error messages
- Verify you're using `http://localhost:5000`

**Styling looks broken?**
- Clear browser cache (Ctrl + F5)
- Check that static files are in correct folders
- Ensure all CSS/JS files are present
